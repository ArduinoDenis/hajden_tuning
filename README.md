# hajden_tuning
Enhanced vehicle tuning script with mechanic job integration

## <b>Requirements</b>
- ox_lib
- ESX (optional, for job restrictions)

## <b>Features</b>
- Framework independent (works with ESX or standalone)
- 0.0ms Usage
- Mechanic job restriction (optional)
- Location-based restrictions (optional)
- Blips for tuning locations
- Improved UI with better icons and feedback

## <b>Installation</b>
- Download hajden_tuning
- Modify the config files:
  - Set `Config.RestrictToMechanics` to true/false based on your preference
  - Set `Config.MechanicJobName` to match your server's mechanic job name
  - Configure tuning locations in `Config.TuningLocations`
- Add hajden_tuning into your resource folder
- Add hajden_tuning into your server.cfg file or ensure hajden_tuning
- Restart server

## <b>Configuration</b>
The script now supports:
- Mechanic job restrictions (ESX integration)
- Location-based tuning (only at designated shops)
- Customizable prices and modification types
- Easy to extend with new features
